# Rock — Paper — Scissors (C)
A small, portable command-line implementation of Rock, Paper, Scissors written in C. Perfect for learning basic I/O, branching, random numbers, and simple game loops.

## Features
* Player vs. computer gameplay.
* Keeps score across rounds. Best of 5.
* Input validation and friendly prompts.

## Requirements
* C compiler (recommended: gcc).
* Standard C library (no external dependencies).

## How to run
* Install a gcc compiler.
* Go to "edit the systems environmental variables"
    * Environment variables
    * Path
    * search for gcc
    * add it to the list
    * restart computer
